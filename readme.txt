Display custom message to your Woocommerce pages to boost your sales.

== Description ==

WooCommerce Custom Notification Display will help you boost your sales by informing your customer about promotions, sales on the shop page, product page, cart page or during checkout process.   

**Features**

- Customizing Template ( live preview )
- Supports PHP 7.2
- Administrative access 
- Add, delete, edit custom message notification
- Select Template for custom message notification
- Hook to Woocommerce pages
- Date publication
